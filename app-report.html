Tutorial

The tutorial can inform the user about the informative resources available on the topic of workplace harassment, how to identify and handle it.

He content is manageable via the CMS.

Logbook

On the logbook, the user can create new entries that can serve as evidence and later on compiled and sent with a report.

If the user has not entered any data yet, the logbook will remain on an empty state.

Report

The user can attach evidence (Logbook), he can either create a new logbook or select an existing one.

The user can send the report right away or save it for later.
